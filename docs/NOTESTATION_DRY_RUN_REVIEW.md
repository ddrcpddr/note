## 是否满足正式导入条件

当前字段质量已经满足“可进入正式导入前人工确认”的条件，但不建议立刻无确认正式导入。

建议正式导入前人工确认：

1. `uncategorized` 是否可以作为默认分类，还是需要建立 Note Station 笔记本到当前分类的映射表。
2. 是否要复制真实附件文件到 `data/attachments/`，以及附件命名规则。
3. 正文是否保留纯文本，还是保留 Note Station HTML / 富文本结构。
4. 是否需要先备份正式 `data/database/app.db`。

# 示例数据说明

项目第一次启动后端或运行检查命令时，会自动创建默认数据。

默认数据来源：

```text
src/shared/defaults.js
```

## 示例成员

默认成员用于模拟家庭多人记录：

- 我
- 爱人

成员名称后续可自定义，例如用户可以改为老婆、妻子、配偶或其他称呼。Note Station 导入记录默认归属执行导入的当前成员，并保留来源元数据。

## 如何重置本地测试数据

停止服务后删除：

```text
data/database/app.db
```

然后运行：

```bash
npm run check
```

Windows PowerShell 可使用：

```powershell
npm.cmd run check
```

注意：删除数据库会清空当前本地记录。真实试用前请先备份整个 `data/` 目录。

## 测试数据不会提交到 GitHub

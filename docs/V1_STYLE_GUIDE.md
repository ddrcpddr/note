不参考 `v2-*` 文件，不继续调用 Product Design，不继续生成 PNG。

## 1. 整体气质

V1 是一个温和、清爽、手机端优先的家庭生活记录 App。它不是后台，不是文档管理器，也不是企业协同系统。

关键词：

- 浅色背景
- 绿色主色
- 圆角卡片
- 轻阴影
- 线性图标
- 底部导航
- 生活感
- 大字号中文
- 信息分组清晰

## 2. 色彩

### 主色

- 主绿色：`#0F8F80`
- 深绿色：`#087267`
- 浅绿色底：`#EAF6F2`

主色用于：

- 页面大标题强调
- 选中状态
- 主按钮
- 重要图标
- 底部导航选中态

### 背景

- 页面背景：`#FCFAF6`
- 卡片背景：`#FFFFFF`
- 浅灰输入底：`#F3F4F1`

背景需要接近白色，但不要纯冷白，整体偏温暖。

### 辅助色

- 橙色：`#D17800`，用于待办、维修、提示
- 红色：`#F02B2B`，用于重要、失败状态
- 蓝色：`#2B76B7`，用于账单、房屋设备等分类
- 紫色：`#7D63B8`，用于导入、证件账号等分类

辅助色以浅底色出现，不能大面积铺满。

## 3. 字体

中文阅读优先，使用系统字体：

```text
Inter, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", "Microsoft YaHei", sans-serif
```

字号建议：

- 首页产品名：40-46px，700
- 页面标题：30-36px，700
- 顶栏标题：22-26px，600
- 卡片标题：22-28px，700
- 正文：17-20px，400
- 辅助信息：14-16px
- 底部导航：13-15px

V1 的文字层级偏大，适合手机阅读。

## 4. 圆角与阴影

- 页面大卡片：18-22px 圆角
- 输入框：18-22px 圆角
- 胶囊筛选：999px 圆角
- 小标签：8-10px 圆角
- 悬浮按钮：圆形或超大圆角

阴影非常轻：

```text
0 8px 24px rgba(39, 43, 48, 0.08)
```

卡片要有柔和浮起感，但不能像商业后台仪表盘。

## 5. 布局

移动端宽度参考 390px / 430px。

- 页面左右边距：18-20px
- 卡片内边距：18-22px
- 卡片间距：12-16px
- 分区标题到卡片：10-14px
- 底部导航高度：72-84px
- 悬浮新建按钮距离底部导航：18-30px

页面纵向滚动，内容不挤。

## 6. 组件规范

### 搜索框

- 高度约 58-66px
- 浅灰底或白底
- 大圆角
- 左侧线性搜索图标
- placeholder 字号偏大

### 筛选胶囊

- 选中态：绿色底 + 白字
- 未选中：白底 + 浅边框 + 深灰字
- 支持图标 + 文案

### 记录卡片

结构：

- 左侧圆形图标背景
- 右侧标题和摘要
- 标签行
- 底部元信息行：时间、分类、附件数量
- 右上角更多按钮

样式：

- 白底
- 大圆角
- 轻阴影
- 分割线很浅
- 标签用浅色底

### 底部导航

四栏：

- 首页
- 分类
- 搜索
- 设置

选中态使用绿色图标和文字。未选中为灰色。底部导航白底、顶部轻边线或轻阴影。

### 悬浮新建按钮

- 首页和分类页右下角出现
- 绿色圆形
- 白色加号
- 轻阴影
- 尺寸约 72px

### 设置页生活插画

V1 设置页顶部有“花瓶 + 画框”生活化装饰。前端原型可以用轻量 CSS/图标模拟生活感，不应变成大幅营销插画。

## 7. 页面风格摘要

### 首页

大标题“家事记”，副标题“记录家里的大小事”。搜索、筛选、今日要记入口、记录卡片和底部导航组成主路径。

### 详情页

顶栏返回、标题、编辑。内容分成：记录摘要卡、内容卡、附件卡、关联记录卡。阅读体验比信息密度更重要。

### 设置页

大标题“设置”，副标题“数据在自己手里更安心”。按数据备份、导出、附件目录、数据库位置、导入入口分组。NAS 信息应该清楚但不技术化。

## 失败的测试项

未发现会阻塞当前 MVP 试用的失败项。

验收过程中发现若干自动化定位文案与真实 UI 不一致，例如新建页标题占位为“标题（可选）”、搜索页占位为“输入关键词”、导入页是“选择文件后自动解析，再确认导入”。这些属于测试脚本定位问题，不是用户功能失败。

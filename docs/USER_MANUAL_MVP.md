# 家事记 MVP 使用手册

这份说明写给家里的普通使用者。你可以把“家事记”当成一个放在家里 NAS 上的家庭记录本，用手机浏览器打开，记录家里的维修、购物、账单、证件、孩子、老人、宠物和临时备忘。

## 首页

首页是家庭生活时间线，最新记录排在最前面。

每张卡片会显示：

- 标题
- 简短摘要
- 时间
- 分类
- 标签
- 创建人
- 附件数量
- 保存状态

可以在首页上方快速筛选：

- 全部 / 待办 / 重要 / 有附件
- 家庭成员
- 常用分类

点击一张记录卡片，可以进入详情页。

## 新建记录

点击底部右侧的绿色圆形按钮，可以新建记录。

可以填写：

- 标题：可不填，系统会用正文自动生成。
- 正文：记录具体内容。
- 记录类型：普通记录、家庭事务、维修维护、购物消费、账号资料、重要备忘、临时想法。
- 标签：例如待办、重要、维修、购物、账单。
- 附件：当前 MVP 只做入口和元数据，后续会支持真实上传。

点击保存后，记录会进入详情页，并保存到本地 SQLite 数据库。

## 详情页

详情页用于完整阅读一条记录。

可以看到：

- 正文
- 分类
- 标签
- 创建人
- 创建时间
- 更新时间
- 来源：手动创建或 Note Station 导入
- 保存状态
- 附件列表

当前编辑能力仍是 MVP 级别，后续会继续完善。

## 搜索

搜索页可以按关键词查找记录。

也可以同时筛选：

- 分类
- 标签
- 成员
- 时间范围

搜索结果仍然以卡片形式显示，适合手机阅读。

## 分类

分类页按家庭生活场景整理记录，例如：

- 家庭事务
- 房屋 / 设备
- 维修 / 售后
- 购物 / 消费
- 证件 / 账号
- 孩子 / 教育
- 老人 / 健康
- 宠物
- 工作 / 杂事
- 临时记录
- 未分类

点击分类可以回到首页查看该分类下的记录。

## 导入 Note Station

设置页里可以进入 Note Station 导入页面。

当前 MVP 使用样例数据演示流程：

1. 选择文件
2. 自动解析预览
3. 确认导入
4. 开始导入
5. 导入完成

真实 Note Station 导入需要后续提供脱敏导出样例后再适配。


## 手机添加到桌面

当前 MVP 已提供基础 PWA manifest。家里局域网服务启动后，可以在手机浏览器中打开应用地址，再使用浏览器菜单里的“添加到主屏幕”或“安装应用”。

注意：当前 MVP 仍需要手机能访问家庭 NAS / 局域网服务，不是完整离线应用。
## 备份

设置页可以查看 NAS 存储与备份状态。

点击“立即备份”会把 SQLite 数据库复制到备份目录。

当前默认备份目录：

```text
data/backups/
```

## 导出 JSON

设置页可以点击“导出 JSON”。

导出的文件会保存到：

```text
data/exports/
```

JSON 导出适合长期归档或以后迁移数据。

## 成员切换

设置页可以切换当前记录人。

新建记录会默认归属到当前成员。

MVP 阶段不做复杂权限，所有家庭成员默认可以查看全部记录。

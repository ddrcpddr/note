# 手机端家庭试运行验收清单

本清单用于我和家人在安卓手机上验收家庭局域网 / NAS 试运行版本。当前版本不继续扩展新功能，重点确认已有 MVP 是否能稳定用于真实家庭记录。

## 试运行前准备

- 确认服务已经在本机、NAS 或 Docker 容器中启动。
- 确认手机与服务所在机器处于同一局域网。
- 优先访问局域网地址，例如：`http://<电脑或NAS局域网IP>:3300`。
- 不要把真实 NAS 地址、账号、密码或 token 写入 Git 文档或配置。

## 安卓手机验收流程

| 序号 | 流程 | 操作 | 通过标准 | 结果 |
| --- | --- | --- | --- | --- |
| 1 | 手机浏览器打开局域网地址 | 在 Chrome / Edge 输入 `http://<局域网IP>:3300` | 首页能打开，不白屏 | 待人工验收 |
| 2 | 添加到桌面 / PWA | 浏览器菜单选择“添加到主屏幕”或“安装应用” | 桌面出现应用图标，点击后可打开 | 待人工验收 |
| 3 | 首页查看记录 | 打开首页，向下滚动记录列表 | 能看到当前正式库记录和导入记录 | 待人工验收 |
| 4 | 新建一条记录 | 点击加号，选择“我”或“爱人”，填写标题和正文并保存 | 保存后进入详情或列表可见 | 待人工验收 |
| 5 | 刷新后记录不丢失 | 浏览器刷新页面或重新打开 PWA | 刚才新建的记录仍存在 | 待人工验收 |
| 6 | 搜索刚才新建记录 | 搜索标题或正文关键词 | 搜索结果能找到该记录 | 待人工验收 |
| 7 | 查看 Note Station 导入记录详情 | 搜索或从未分类进入一条导入记录 | 能看到标题、正文、来源、时间、原始路径和附件元数据 | 待人工验收 |
| 8 | 分类筛选 | 打开分类页，进入“未分类 / 待整理”或其他分类 | 能看到对应记录，页面不跳错 | 待人工验收 |
| 9 | 成员筛选 | 在首页或搜索页切换“全部 / 我 / 爱人” | 筛选结果符合当前成员 | 待人工验收 |
| 10 | 设置页手动备份 | 进入设置，点击“立即备份” | 页面提示成功，NAS / data/backups 下出现备份文件 | 待人工验收 |
| 11 | JSON 导出 | 设置页点击“导出 JSON” | 页面提示成功，NAS / data/exports 下出现 JSON 文件 | 待人工验收 |
| 12 | NAS 离线或服务不可用提示 | 临时停止服务或使用页面里的离线测试入口 | 提示能看懂，不误导为已保存 | 待人工验收 |
| 13 | 长标题 / 长正文 / 长链接 | 打开含长 URL 或长正文的记录 | 不横向溢出，不遮挡按钮 | 待人工验收 |
| 14 | 底部导航遮挡检查 | 首页、搜索、分类、设置页滚动到底部 | 底部导航不挡住关键按钮和正文 | 待人工验收 |
| 15 | 添加到桌面后的启动体验 | 从桌面图标启动应用 | 首屏能正常加载，返回/刷新体验可接受 | 待人工验收 |

## 建议记录的问题

人工验收时如果发现问题，建议记录：

- 手机型号和浏览器。
- 访问地址。
- 出问题的页面。
- 操作步骤。
- 是否能稳定复现。
- 页面截图，截图中不要包含真实隐私正文。

## 当前不在本轮验收范围

- 新增成员。
- 复杂权限系统。
- 外网访问、内网穿透或反向代理。
- 安卓原生 App 封装。
- 复杂离线同步。

# Image Asset Prompts

These are prompt drafts for a future image2 Agent. Do not run them in this Codex turn.

## Global Negative Prompt

Do not include real note text, real names, real NAS addresses, passwords, tokens, QR codes, real documents, invoices, medical records, cyberpunk style, high-saturation tech blue, dark sci-fi UI, enterprise dashboard panels, or full-page app UI.

## 1. PWA App Icon

Purpose: app icon for home-screen install.

Prompt:

```text
Create a warm mobile app icon for a family life record app named 家事记. Use a simple rounded notebook or home-shaped note symbol, soft green primary color, warm off-white background, gentle household feeling, minimal vector style, clear at small sizes, no readable private text, no dashboard or tech-cyber style.
```

Suggested output:

- 1024 x 1024.
- Rounded-square safe area.
- No tiny details.

## Prompt Hygiene

Before generation, the future image2 Agent must verify:

- The prompt does not ask for full UI screens.
- The prompt does not include real family content.
- The prompt does not include real imported note content.
- The prompt does not include a NAS address or credential.
- The prompt keeps V1 warm family style.
